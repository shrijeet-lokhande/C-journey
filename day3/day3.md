## Instructions

There are following types of instructions in the C
```text
Instructions
   |
   |---Type declaration instructions
   |
   |---Arithematic instructions
   |
   |---Control instructions
   ```

   ### <u>_1.Type declaration Instructions_</u>
instructions that are used to declare the variable and specify their data type are known as ``` type declaration instructions ```.

### <u>_2.Control Instructions_</u>
Instructions that are used to control the flow of execution of the program are knwon as ```control instructions```.

### <u>_Arithematic Instructions_</u>
Instructions that are used to perform mathematical operations on the Operands are knwon as ```Arithematic instructions```.

## Conversion

```
Conversion
   |
   |---implicit conversion
   |
   |---explicit conversion
   ```
   - <u>Implicit Conversion</u> :- Done by the compiler while compiling the code.
   - <u>Explicit Conversion</u> :- Has to be specified by the programmer while coding the program.
